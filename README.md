RESTful API для управления базой данных фильмов, отзывов, пользовательских профилей и персональных подборок.
🚀 Основные возможности
  🎬 Фильмы и Жанры
    CRUD фильмов: Полный цикл управления фильмами (создание, чтение, обновление, удаление) через /movies.
    Фильтрация: Поиск фильмов по жанру и году выпуска.
    Жанры: Управление справочником жанров через /genres.
    
  💬 Отзывы и Рейтинги
    Система отзывов: Возможность оставлять отзывы к фильмам и просматривать их.
    Авто-рейтинг: Средний рейтинг фильма (avg_rating) автоматически пересчитывается при добавлении новых отзывов.
    
  👤 Пользователи и Watchlist
    Авторизация: Регистрация, логин и просмотр профиля пользователя.
    Список желаемого (Watchlist): Добавление фильмов в персональный список "Посмотреть позже".
    
  📁 Подборки (Collections)
    Пользователи могут создавать тематические списки фильмов (например, "Лучшие ужасы" или "Фильмы 2024").
    
🛠 Эндпоинты API
  Фильмы
    Метод	Эндпоинт	Описание
    GET	/movies	Получить список фильмов (доступна фильтрация ?genre_id=1&year=2024)
    POST	/movies	Добавить новый фильм
    PUT	/movies/:id	Обновить данные фильма
    DELETE	/movies/:id	Удалить фильм
    
  Отзывы и Жанры
    Метод	Эндпоинт	Описание
    POST	/movies/:id/reviews	Оставить отзыв к фильму
    GET	/movies/:id/reviews	Посмотреть все отзывы к фильму
    GET/POST	/genres	Получить список или добавить новый жанр
    
  Аккаунт и Watchlist
    Метод	Эндпоинт	Описание
    POST	/register	Регистрация нового пользователя
    POST	/login	Вход в систему
    GET	/users/:id	Профиль пользователя
    POST/DELETE	/watchlist	Управление списком "Посмотреть позже"
    GET	/users/:id/watchlist	Просмотр вачлиста конкретного пользователя
    
  Тематические подборки
    Метод	Эндпоинт	Описание
    POST	/collections	Создать новую подборку
    GET	/collections	Список всех подборок
    GET	/collections/:id	Получить конкретную подборку
    POST	/collections/:id/movies	Добавить фильм в подборку
    DELETE	/collections/:id/movies/:movie_id	Удалить фильм из подборки

🛠 Технологии
Language: Go 
Database: PostgreSQL
